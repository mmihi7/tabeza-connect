Tabeza Connect Usage
--------------------

Tabeza Connect bundles and uses clawPDF as-is without modifications. We:
- Bundle the official clawPDF 0.9.3 MSI installer
- Install it silently during Tabeza Connect installation
- Configure it via registry settings (not code modifications)
- Provide attribution and license information to end users
