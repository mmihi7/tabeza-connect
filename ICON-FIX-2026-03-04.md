# Icon Fix - March 4, 2026

## Problem
System tray icon was not appearing after installation.

## Root Cause
- App was looking for `resources/icon.ico`
- Should be using `assets/icon-green.ico` (the original icon)

## Changes Made

### 1. Updated package.json
```json
{
  "build": {
    "directories": {
      "buildResources": "assets"  // Changed from "resources"
    },
    "files": [
      "assets/**/*"  // Changed from "resources/**/*"
    ],
    "win": {
      "icon": "assets/icon-green.ico"  // Changed from "resources/icon.ico"
    },
    "nsis": {
      "installerIcon": "assets/icon-green.ico",
      "uninstallerIcon": "assets/icon-green.ico",
      "installerHeaderIcon": "assets/icon-green.ico"
    }
  }
}
```

### 2. Updated electron-main.js
```javascript
// Changed RESOURCES_PATH for development
const RESOURCES_PATH = isDev 
  ? path.join(__dirname, '../assets')  // Changed from '../resources'
  : process.resourcesPath;

// Changed icon filename in all locations
function createTray() {
  const iconPath = path.join(RESOURCES_PATH, 'icon-green.ico');  // Changed from 'icon.ico'
  // ...
}

// Updated all window icons
icon: path.join(RESOURCES_PATH, 'icon-green.ico')  // Changed from 'icon.ico'

// Updated notification icon
icon: path.join(RESOURCES_PATH, 'icon-green.ico')  // Changed from 'icon.ico'
```

### 3. Deleted old resources folder
- Removed `resources/icon.ico`

## Expected Result
✅ System tray icon appears with green Tabeza icon  
✅ Installer uses green icon  
✅ All windows use green icon  
✅ Notifications use green icon
